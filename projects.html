furcare-app.fig
UI/UX Case Study

FurCare — Pet Care App

An appointment booking & pet-health app that made vet visits feel less like an emergency and more like routine care.

Problem

Pet owners abandoned bookings mid-flow because vet availability and pricing were buried three screens deep.

Solution

Surfaced live vet availability and price upfront, and added a one-tap "emergency" path separate from routine bookings.

Process

User interviews with 8 pet owners → journey mapping → low-fi wireframes → hi-fi prototype → 2 rounds of usability testing.

Tools

Figma, FigJam, Maze for usability testing.

UI/UX Case Study

WhatsApp UI Redesign

A concept redesign focused on decluttering chat threads and making media sharing feel faster on low-end Android devices.

Problem

Chat list and media viewer felt cluttered on smaller screens, with unclear unread-state hierarchy.

Solution

Reworked message bubbles, unread badges and a lighter media viewer with clearer gesture affordances.

Process

Heuristic evaluation of the existing app → competitive audit → component-level redesign → animated prototype.

Tools

Figma, Principle for micro-interaction previews.

irctc-railconnect.fig
UI/UX Case Study

IRCTC RailConnect Redesign

Reworking one of India's busiest booking flows so a return ticket search takes fewer taps than opening the seat chart.

Problem

Search, seat availability and payment were spread across disconnected screens with inconsistent visual hierarchy.

Solution

Combined search + availability into one scrollable results screen, with a persistent fare summary bar.

Process

Task-flow audit → card sorting for information architecture → wireframes → clickable prototype tested with 6 frequent travellers.

Tools

Figma, Google Forms for the traveller survey.

where-is-my-train.fig
UI/UX Case Study

Where Is My Train Redesign

Live-tracking app redesign prioritizing the one thing a stressed passenger actually wants: "how many minutes until my train arrives."

Problem

The most-needed number (time to arrival) was smaller and less prominent than decorative map elements.

Solution

Redesigned the home screen around a single large ETA readout, with the map demoted to a secondary tab.

Process

Contextual inquiry at a railway platform → priority guides → wireframes → hi-fi UI kit.

Tools

Figma, on-site field notes.

ecommerce-app.fig
UI/UX Case Study

E-commerce App Design

A ground-up shopping app concept built around fast browsing and a frictionless one-page checkout.

Problem

Reference apps buried "add to cart" behind multiple taps and reset filters on every back navigation.

Solution

Sticky filters, quick-add from the grid view, and a single-screen checkout with saved details.

Process

Competitive teardown of 4 shopping apps → moodboard → component design system → full prototype.

Tools

Figma, Figma Community plugins for icons.
